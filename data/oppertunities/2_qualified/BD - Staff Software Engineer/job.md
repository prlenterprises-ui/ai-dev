• About Staff Software Engineer Role

Architect, design, and develop a cloud-based clinical interoperability platform using Go, built on AWS infrastructure. Ensure the platform strictly complies with healthcare interoperability standards such as HL7 and FHIR, and regulations like HIPAA. Drive best practices in cloud architecture, including serverless and containerization approaches using AWS Lambda, ECS, and Fargate. Implement CI/CD pipelines using AWS CodePipeline, CodeBuild, and CodeDeploy for continuous integration and deployment. Lead and mentor a team of engineers, ensuring best practices in AWS development, cloud security, and infrastructure-as-code (Terraform/CloudFormation).
